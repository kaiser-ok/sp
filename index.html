Network line quality test
Test method :
Test 20 times in sequence with HTTP Method HEAD (each test timeout is 3 seconds), and the total test time will be terminated immediately if it exceeds 5 seconds), calculate the average response time (ms), and use colors to indicate its status. If the endpoint does not support the HTTP communication protocol, it will display "No data"
